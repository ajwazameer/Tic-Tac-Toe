# Tic Tac Toe

A browser-based Tic Tac Toe game built with HTML, CSS, and JavaScript. This project demonstrates fundamental front-end development concepts including DOM manipulation, event handling, game state management, and win-condition logic.

## Live Demo

https://ajwazameer.github.io/tic-tac-toe/

---

## Features

- Two-player gameplay (X vs O)
- Prevents overwriting occupied cells
- Detects winning combinations automatically
- Detects tie games
- Highlights the winning combination
- Reset functionality
- Responsive user interface

## Game Rules

1. Player **O** starts first.
2. Players alternate turns placing their symbol.
3. The game checks for a winner after every move.
4. If three identical symbols align horizontally, vertically, or diagonally, the winner is declared.
5. If all cells are filled without a winner, the game ends in a tie.
6. The Reset button starts a new game.
